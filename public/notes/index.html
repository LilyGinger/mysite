如何获取 Token？
  1. 访问 GitHub Token 创建页面
  2. Note填写：My Notes App
  3. Expiration 选择：No expiration
  4. 勾选 gist 权限
  5. 点击 Generate token并复制
  6. 粘贴到上方输入框并点击连接
